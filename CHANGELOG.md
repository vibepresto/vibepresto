# Changelog

## 0.1.1

- Added a public `/meta` endpoint and compatibility metadata in `/auth/me` for CLI and skill version checks.
- Declared minimum supported CLI and skill versions in the plugin as the compatibility source of truth.
- Added framework-aware static deployment support with route manifests and multi-route page mapping.
- Expanded the plugin API for deployments, batch page resolution, and richer bundle metadata.
- Improved the renderer and compatibility flow so single-page uploads continue to work alongside multi-page deployments.
